# Научиться учиться.

Первый проект статической верстки на курсе "Веб-разработчик плюс" от Яндекс Практикум.

В течение месяца, студенты изучают лекционный материал и прорабатывают его в тренажерах, на платформе Яндекс Практикума. После изучения учебного материала, студент выполняет итоговую проектную работу и сдает на проверку ревьюеру.

## Проектная работа.

Проектная работа состоит из 2-ух этапов:

[Первый этап](https://code.s3.yandex.net/web-plus/project-1/sprint-1-brief.pdf) выполняется после изучения: структуру файлов, основу семантики HTML, метадалогию БЭМ, позиционирование, флексбоксы.

[Второй этап](https://code.s3.yandex.net/web-plus/project-1/sprint-2-brief.pdf) выполняется после изучения: API, iframe, embed, видео в HTML, анимация и трансформация.

В финале работа должна получиться [примерно такой](https://code.s3.yandex.net/web-developer/final-projects/project-1/index.html)
